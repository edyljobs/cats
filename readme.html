# ## Description A demo project to be submitted to IONA ## Installation Step 1 : Download and install docker at https://www.docker.com/ Step 2 : Clone or Download this git repo and extract to your device Step 3 : Run the docker compose file found on the root of the extracted file : docker-compose up -d Step 4 : open google chrome and go to http://localhost:8080/ (user: wordpress , password : wordpress) if wordpress table exist drop the table and import the wordpress.sql file found on the root directory of this folder Step 5 : open google chrome and go to http://localhost:8000 You can now start using the demo project.
